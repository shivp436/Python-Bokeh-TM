### Python Bokeh Data Visualisation

> Chart/visualization of the top horsepower cars. Created using Python and the Bokeh library. This is part of the data visualization YouTube tutorial

### Quick Start
``` 
# Install dependencies
pipenv install

# Generate html
python main.py
```
